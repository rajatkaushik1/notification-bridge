Method 1: MacroDroid (Android App - No Code Required)

Trigger instant laptop notifications when your phone rings using the free MacroDroid automation app.

  1. Install MacroDroid from the Google Play Store.
  2. Tap Add Macro.
  3. Trigger: Tap Call/SMS → Call Incoming → Select Any Number (or specific contacts).
  4. Action: Tap Web Interactions → HTTP Request.
  5. Set Request Method to POST and enter your server URL:
    http://YOUR_LAPTOP_IP:3000/api/call
  6. In Content Type choose application/json and paste this JSON Body: 
    {
  "number": "[call_number]",
  "contactName": "[call_name]",
  "status": "Ringing",
  "tag": "General"
}
  7. Save the macro. Now whenever a call arrives, your laptop rings and saves the call record!

Method 2: Tasker or Automate (Android)

If you prefer Tasker or Automate, configure an HTTP POST task triggered by Phone Ringing event.

  • URL: http://YOUR_LAPTOP_IP:3000/api/call
  • Method: POST
  • Headers: Content-Type: application/json
  • Body: {"number": "%CNUM", "contactName": "%CNAME"}

Method 3: Native Python Desktop Script

Keep laptop_receiver.py running in your terminal to receive native OS desktop popup notifications even when your browser is closed:

# Start Python desktop notification receiver
python laptop_receiver.py

# View local call log in terminal
python laptop_receiver.py --history
