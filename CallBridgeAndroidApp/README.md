# CallBridge PRO — Your Custom Native Android App

This folder contains your **very own native Android App** written in Kotlin (`CallBridge PRO`). You own 100% of the code—no ads, no paywalls, no third-party automation apps needed ever again!

## ✨ Features
1. **Background Call Receiver (`CallReceiver.kt`):** Automatically detects incoming phone calls and sends an HTTP POST request to your Cloudflare Webhook.
2. **Sleek Control Panel UI (`MainActivity.kt`):**
   - Toggle call listening ON/OFF
   - Customize your Cloudflare Webhook URL
   - **Send Test Call to Laptop** button to verify instant connectivity

## 🚀 How to Build Your APK
1. Open this folder (`CallBridgeAndroidApp`) in **Android Studio**.
2. Click **Build → Build Bundle(s) / APK(s) → Build APK(s)**.
3. Transfer the generated `app-debug.apk` to your Android phone and install it!
4. Open **CallBridge PRO**, grant Phone State permission, and tap **"Send Test Call to Laptop"**!
